# Neural Network Homework Execution Guide

Author: <Your Name>

This project implements neural networks for regression and classification
using both a custom neural network framework and PyTorch.

-----------------------------------------------------

# 1 Install Conda

Install Miniconda or Anaconda:

https://docs.conda.io/en/latest/miniconda.html

-----------------------------------------------------

# 2 Create Environment

Navigate to project folder and run:

conda env create -f environment.yml

-----------------------------------------------------

# 3 Activate Environment

conda activate nn_homework

-----------------------------------------------------

# 4 Start Jupyter

jupyter notebook

-----------------------------------------------------

# 5 Run Notebooks

Navigate to folder:

notebooks/

Run notebooks in this order:

1 regression_from_scratch.ipynb
2 binary_classification_from_scratch.ipynb
3 multiclass_classification_from_scratch.ipynb
4 regression_pytorch.ipynb
5 binary_classification_pytorch.ipynb
6 multiclass_classification_pytorch.ipynb

-----------------------------------------------------

# 6 Expected Output

Each notebook will show:

- training loss curves
- model evaluation metrics
- classification accuracy

-----------------------------------------------------

# 7 Hardware Requirements

CPU: standard laptop CPU  
RAM: >= 8GB recommended  

No GPU required.
